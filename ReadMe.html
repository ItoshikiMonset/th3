To install the script, please follow the steps 1-4
   
1. 
  
To install, simply upload and unzip the file rapidleech.zip in the following folder of your cPanel account:

/home/YOUR_CPANEL_USERNAME/public_html/
     
2. 
  
Then check that the following files have permissions 0744:

/home/YOUR_CPANEL_USERNAME/public_html/rar/rar

/home/YOUR_CPANEL_USERNAME/public_html/rar/unrar
   
3. 
  
The following folder permissions 0777:

/home/YOUR_CPANEL_USERNAME/public_html/mtn/
   
4. 
  

And the following file permissions 0777:

/home/YOUR_CPANEL_USERNAME/public_html/mtn/mtn


Default login details:
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

User: rapidleech
Pass: host

To change default login details edit file:

/home/YOUR_CPANEL_USERNAME/public_html/configs/config.php

Search:

'users' =>
  array (
    'rapidleech' => 'host',
  ),

Edit and save changes.
 
  

 

Premium Accounts:
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

To be able to use your own premium accounts, edit file:

/home/YOUR_CPANEL_USERNAME/public_html/configs/accounts.php

Delete the comment '//' at the beginning of the line, then replaces 'your username' and 'your password' with your login details, e.g.

$premium_acc["uploaded_net"] = array('user' => '8746532', 'pass' => 'k9o0y4d6');

Only edit each file server settings you need to use and saves changes.

 
  

 

Prohibited Files:
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

To be able to work with prohibited files (required for PC Uploader, Movie Thumbnailer, Video Player), edit file:

/home/YOUR_CPANEL_USERNAME/public_html/.htaccess

Search:

<files ~ "\.(?i:s?php.*|cgi|p[ly]|sh|asp|[dsp]?html?)$">
deny from all
allow from 91.15.171.199
</files>

Replace with your IP address, e.g.

<files ~ "\.(?i:s?php.*|cgi|p[ly]|sh|asp|[dsp]?html?)$">
deny from all
allow from 65.154.58.124
</files>

Edit and save changes. If you do not know your IP address:
